## 👻 분기문

### 🌱 Branch
이전에도 한 번 나왔었지만 특정 조건에 따라 참/거짓으로 길을 나눠주는 노드이다. ``` branch ``` 검색도 가능하고, 단축키 ``` B + 좌클릭 ```으로도 생성 가능하다.
